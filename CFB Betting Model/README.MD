# CFB Betting Model

Project using Python to wrangle data and R to create and assess different models for predicting the outcomes of college football games against the spread.

## Libraries
* Python
  * Pandas
  * Numpy 
* R
  * dplyr
  * tidyverse 
  * ggplot2
  * car
  * readr
  * corrplot
  * Hmisc
  * QuantPsyc
  * PerformanceAnalytics
